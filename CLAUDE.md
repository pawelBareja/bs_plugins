# Claude Instructions

At the start of every session, read the following files from the `.claude/` folder:

- `.claude/plugins.instructions.md` — block naming conventions, project structure, and rules you must follow
- `.claude/coding.practices.md` — general coding standards for all source files
- `.claude/react.instructions.md` — React and hooks standards for `.tsx` files
- `.claude/styles.instructions.md` — SCSS styling standards
